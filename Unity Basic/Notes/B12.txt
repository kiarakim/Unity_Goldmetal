[게임 인터페이스. UGUI 기초]

1. 캔버스
Canvas : UI가 그려지는 도화지 역할인 컴포넌트

2. 스크린
스크린 : 게임이 표시되는 화면. 해상도로 크기 결정
마우스 커서도 스크린 좌표계에 포함.

3. 텍스트 UI
Text : 문자열을 표시하는 UI
Canvas > UI > Text -> 폰트는 꼭 라이센스 확인

4. 이미지 UI
Image : 이미지를 표시하는 UI
이미지 파일을 Sprite로 설정해야 UI 적용 가능
이미지가 깨져보인다면 비율을 맞춰야함.
 ㄴPreserve Aspect : 비율 고정. 체크
 ㄴSet Native Size : 실제크기로 맞춰줌
UI는 Hierarchy의 순서에 따라 위 아래 결정

5. 버튼
Sliced 기능으로 자유자재 크기 이미지 구현 가능
OnClick() : 버튼 클릭 시 호출되는 이벤트 함수
버튼 클릭은 '누르기-떼기' 한 세트 동작

6. 앵커
앵커-빨간점 : 캔버스에서의 기준점
앵커-파란점 : 컴포넌트에서의 기준점(Shift)
앵커-중앙네모 : 컴포넌트에서의 위치(Alt)
